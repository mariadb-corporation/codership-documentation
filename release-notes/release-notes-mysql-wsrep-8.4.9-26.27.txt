# Changes in MySQL-wsrep 8.4.9-26.27 (2026-05-13)

MariaDB is pleased to announce the first LTS release of Galera Cluster for
MySQL 8.4, consisting of MySQL-wsrep 8.4.9 and wsrep API version 26.

This release incorporates all changes up to MySQL 8.4.9.

Galera Cluster 8.4 is available as targeted packages and package repositories
for Debian 11 (Bullseye), and 12 (Bookworm), Ubuntu 22.04 LTS
(Jammy), and Ubuntu 24.04 LTS (Noble), RHEL 8, and 9. Obtaining packages using a
package repository removes the need to download individual files and facilitates
the deployment and upgrade of Galera nodes.

This and future releases will be available from https://mariadb.com/galera-downloads/.

Notable changes and fixes in MySQL-wsrep in 8.4.9-26.27:

- Added --galera-info parameter mysqldump tool to add wsrep start position
to the end of database dump.

Known Issues:

- InnoDB tablespaces outside of the data directory are not supported, as they
may not be copied over during SST.
